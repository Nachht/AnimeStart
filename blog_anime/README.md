# 🌸 Anime Blog

Blog web desarrollado como proyecto colaborativo con el objetivo de compartir contenido relacionado con el mundo del anime. La aplicación permite explorar publicaciones, navegar por diferentes secciones y ofrecer una experiencia atractiva para los usuarios interesados en la cultura anime.

---

## 🚀 Características

- 📰 Diseño tipo blog.
- 🎌 Secciones dedicadas al mundo del anime.
- 📱 Diseño adaptable (Responsive).
- 🎨 Interfaz desarrollada con HTML y CSS.
- ⚡ Funcionalidades dinámicas implementadas con JavaScript.
- 🤝 Proyecto desarrollado en equipo.

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## 💻 Instalación

1. Clona el repositorio.

```bash
git clone https://github.com/TU-USUARIO/Anime-Blog.git
```

2. Entra en la carpeta.

```bash
cd Anime-Blog
```

3. Abre el archivo `index.html` en tu navegador.
